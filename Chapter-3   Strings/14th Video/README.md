## Chapter 3 — Strings

### What I learned
- What Strings are in Java and how to declare them
- String is not a primitive type — it is a class in Java
- How to take String input from the user using Scanner
- Important String methods:
  - `length()` — gives the number of characters
  - `charAt()` — gives character at a specific position
  - `toUpperCase()` and `toLowerCase()` — converts case
  - `trim()` — removes extra spaces
  - `substring()` — extracts part of a string
  - `contains()` — checks if a string has certain text
  - `equals()` — compares two strings

### Key concepts
- Never use `==` to compare Strings — always use `.equals()`
- Strings in Java are immutable — once created they cannot be changed
- String indexing starts from 0, not 1
