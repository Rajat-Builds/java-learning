## Chapter 1 — Variables and Data Types

First chapter of my Java learning journey.

### What I learned
- What variables are and how to declare them in Java
- All primitive data types — int, float, double, char, boolean, long, byte, short
- The difference between integer and decimal numbers in Java
- How to store and print different types of data
- Basic type casting — converting one data type to another

### Key concepts
- Every variable in Java must have a type
- Java is a strongly typed language — you can't store text in an integer variable
- `int` for whole numbers, `double` for decimals, `char` for single characters, `boolean` for true/false
