## Chapter 4 — Conditional Statements

### What I learned
- How to make decisions in Java using conditions
- if, if-else, and if-else-if ladder
- Nested if-else — putting conditions inside conditions
- Switch statement — cleaner way to handle multiple fixed conditions
- Ternary operator — shorthand for simple if-else in one line

### Key concepts
- The condition inside if() must always return true or false
- Switch works best when comparing one variable against fixed values
- Ternary operator syntax: `condition ? valueIfTrue : valueIfFalse`
- Java executes only the first matching condition in an if-else-if ladder
- Always add `break` inside switch cases to prevent fall-through
