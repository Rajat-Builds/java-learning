## Chapter 2 — Operators and Expressions

### What I learned
- Arithmetic operators — addition, subtraction, multiplication, division, modulus
- Assignment operators — =, +=, -=, *=, /=
- Comparison operators — ==, !=, >, <, >=, <=
- Logical operators — && (AND), || (OR), ! (NOT)
- Increment and decrement — i++ and i--
- How Java evaluates expressions using operator precedence

### Key concepts
- The modulus operator `%` gives the remainder of a division
- `==` checks equality, `=` assigns a value — very common mistake for beginners
- Logical operators are used to combine multiple conditions
- Pre-increment `++i` and post-increment `i++` behave differently
