## Chapter 5 — Loop Control Instructions

### What I learned
- How to repeat a block of code using loops
- for loop — when you know exactly how many times to repeat
- while loop — when you repeat until a condition becomes false
- do-while loop — runs at least once before checking the condition
- Nested loops — loops inside loops
- break — exits the loop immediately
- continue — skips the current iteration and moves to the next

### Key concepts
- for loop is best when the number of iterations is known
- while loop is best when the condition depends on user input or runtime value
- do-while guarantees the code runs at least one time
- Infinite loops happen when the condition never becomes false — always double check
- Nested loops are commonly used for patterns and 2D data like matrices
