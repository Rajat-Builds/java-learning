## Chapter 7 — Methods In Java

### What I learned
- What methods are and why we use them
- How to define and call a method in Java
- Methods with parameters — passing data into a method
- Methods with return types — getting data back from a method
- void methods — methods that do a task but return nothing
- Method overloading — same method name with different parameters
- Recursion — a method that calls itself to solve a problem

### Key concepts
- Methods help avoid repeating the same code — write once, use many times
- Every method must declare what type of value it returns
- If a method returns nothing, use `void` as the return type
- Method overloading is decided at compile time based on parameters
- Recursion must always have a base case — otherwise it runs forever
- Each recursive call adds a new layer to the call stack
