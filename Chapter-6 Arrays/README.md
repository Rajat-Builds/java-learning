## Chapter 6 — Arrays

### What I learned
- What arrays are and why we use them
- How to declare, initialize and access arrays in Java
- Single dimensional arrays — storing a list of values
- Multi dimensional arrays — storing data in rows and columns
- Iterating through arrays using for loop and for-each loop
- Finding the length of an array using `.length`
- Common array operations — finding max, min, sum, and reversing

### Key concepts
- Arrays in Java have a fixed size — once created the size cannot change
- Array indexing starts from 0 — first element is at index 0
- Accessing an index that doesn't exist causes `ArrayIndexOutOfBoundsException`
- For-each loop is cleaner for reading arrays but can't modify elements
- Multi dimensional arrays are arrays of arrays — `int[][] matrix`
